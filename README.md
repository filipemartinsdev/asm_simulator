# ⚙ Assembly Simulator

Pacote especializado para simular Assembly x86_32 em alto nível com Java.

> Em construção...

## Sobre

Esse projeto visa implementar em alto nível vários dos comportamentos do ASM, permitindo a compreensão do básico da linguagem.

## Features

### Alocação de memória
- Implementada puramente em arrays de bytes; 
- Utilização de operações bitwise para manipulação de dados.
- Instruções DB e RESB;

### Instruções gerais
- MOD, ADD, SUB

### Mais
- Registradores de 32Bits (EAX, EBX, ECX, EDX) e suas respectivas separações (AL, AH, BL, BH...);
- Syscalls básicas
